# CORE AI — Frontend Summary Notes
# Prepared for Presentation
# ============================================================

## PROJECT OVERVIEW
Core AI is a Gemini-style AI chat dashboard built with:
- React + Vite (Frontend)
- Express + Socket.IO (Backend)
- MongoDB (in-memory via mongodb-memory-server for local dev)
- Framer Motion (animations)
- Lucide React (icons)
- TailwindCSS (styling)

# ============================================================
## 1. AUTHENTICATION PAGES
# ============================================================

FILES:
  - Frontend/src/Auth/SignIn.jsx
  - Frontend/src/Auth/SignUp.jsx

FEATURES:
  - Workspace Registration (SignUp) with multi-field form
  - Email/Password login (SignIn)

# ============================================================

FILE: Frontend/src/Dashboard/components/AiLayout.jsx

FEATURES:

  - Both modes use translateX (not width) for GPU-accelerated smoothness

## 4. DASHBOARD HOME / CHAT PAGE (Home.jsx)
# ============================================================

FILE: Frontend/src/Dashboard/pages/Home.jsx

FEATURES:
  - Greeting: "What can I help with, {FIRSTNAME}?" (name from localStorage)
  - Real-time AI chat via Socket.IO WebSocket
  - Streaming responses (chunk-by-chunk like Gemini)
  - Loading skeleton (3 bouncing dots) while AI responds
  - Custom Markdown renderer:
      * Bold, inline code, code blocks (with Copy button)
      * Tables, headers (H1/H2/H3), bullet lists, horizontal rules

PROMPT BOX (Gemini-style):
  - Container: rounded-[24px] with bg-[#f0f4f9] (light) / bg-[#1e1f20] (dark)
  - Auto-expanding textarea (grows up to 200px height)
  - Plus (+) button on left:
      * Rotates 45deg when attachment menu opens (spring animation)
      * whileTap scale: 0.85 for tactile feedback
  - Attachment popup menu: Upload files, Add from Drive, Create image, Enhance image, Create video
    * Closes automatically when clicking anywhere outside the menu
  - Right side (idle): Image icon + Mic icon
  - Right side (typing): Send button (ArrowUp, high-contrast bg)
  - Enter to send, Shift+Enter for new line

  npm run dev
  -> http://localhost:5173

Backend:
  cd Backend
  node src/server.js
  -> http://localhost:5000
  (MongoDB starts automatically, no installation needed)
